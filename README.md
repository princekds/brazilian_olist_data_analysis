# 📊 Olist Brazilian E-Commerce Data Analysis Project

## 📌 Overview

This project is an end-to-end data analytics case study on the **Olist Brazilian E-Commerce dataset**. It explores customer behavior, sales performance, delivery efficiency, and product trends using Python, PostgreSQL, and Power BI.

The goal is to extract meaningful business insights and present them through data analysis and interactive dashboards.

---

## 🎯 Objectives

- Clean and preprocess raw multi-table dataset
- Perform exploratory data analysis (EDA)
- Merge and structure datasets for analysis
- Store and analyze data using PostgreSQL
- Write SQL queries for business insights
- Build an interactive Power BI dashboard

---

## 🧰 Tech Stack

- Python (Pandas, NumPy, Matplotlib, Seaborn, Plotly)
- PostgreSQL (Database & SQL analysis)
- Power BI (Data visualization & dashboarding)
- Jupyter Notebook

---

## 📂 Dataset

The dataset is from **Olist**, a Brazilian e-commerce platform and contains multiple relational tables such as:

- Orders
- Customers
- Products
- Sellers
- Payments
- Reviews
- Geolocation

(Source: Kaggle Olist dataset)

---

## 🔄 Project Workflow

### 1. Data Cleaning & Preprocessing (Python)
- Handled missing values
- Removed duplicates
- Standardized column formats
- Merged multiple datasets into a unified structure

### 2. Exploratory Data Analysis (EDA)
- Sales trends over time
- Customer distribution across regions
- Product category performance
- Revenue patterns
- Delivery time analysis

### 3. SQL Analysis
- Designed relational database schema
- Loaded cleaned dataset into PostgreSQL
- Executed SQL queries for insights:
  - Monthly revenue trends
  - Top product categories
  - Customer distribution by state
  - Delivery time vs review score relationship

### 4. Power BI Dashboard
- Built interactive dashboard with KPIs
- Sales and revenue trends visualization
- Customer segmentation insights
- Product performance analysis
- Delivery efficiency metrics

---

## 📊 Key Insights

- A small number of product categories generate most of the revenue
- Delivery delays negatively impact customer review scores
- Certain regions in Brazil dominate order volume
- Clear seasonal patterns exist in sales trends
- Payment preferences vary across customer segments
